## Dependencies
* PHP 7.1
* MariaDB 10.2
* Laravel 5.5 [Requirements](https://laravel.com/docs/5.5#installation)
* Yarn
* Redis

## Dev
* `git clone https://github.com/kbkyzd/shiraishi` - Clone the repo
* `composer install` - Install PHP dependencies
* `yarn` - Node tooling dependencies
